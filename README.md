# Succulent dotfiles

## Included programs:

- dwm
- st
- dmenu
- dwmblocks
- neovim
- zsh
- picom
- tmux
- redshift

## How to use the setup script

NOTE: Setup script is currently only compatible with Void Linux

1. Clone the repo
```shell
git clone --depth 1 https://github.com/aidanhopper/dotfiles --recurse-submodules .dotfiles
```
2. Run the setup script
```shell
./setup
```
